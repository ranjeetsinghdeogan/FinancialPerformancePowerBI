# Financial Performance Analysis Dashboard (Power BI)

## Project Overview

This project focuses on analyzing financial data to understand profit, revenue trends, and sales performance across different segments and countries. The objective was to build an interactive Power BI dashboard that helps visualize financial performance and support business decision-making.

## What I Did

* Imported and transformed financial data in Power BI
* Cleaned and formatted date and currency fields
* Analyzed total profit by segment and country
* Created visualizations to track revenue and profit margin trends over time
* Built charts to show units sold and discount patterns
* Added slicers to filter data dynamically by Segment, Country, and Year

## How I Did It

* Used Power BI for data modeling and visualization
* Applied data transformation using Power Query
* Created measures to calculate profit and profit margin
* Designed advanced visuals such as waterfall and ribbon charts
* Implemented slicers to enable interactive dashboard filtering

## Result

* Successfully built an interactive financial dashboard
* Identified profit trends across countries and customer segments
* Visualized monthly and yearly financial performance
* Analyzed the impact of discounts on sales and profit
* Demonstrated practical skills in data visualization and business analysis

* Power BI
* Power Query
* Data Visualization
* Dashboard Design
